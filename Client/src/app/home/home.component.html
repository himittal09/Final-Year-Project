Exam Analysis System


A place for all students to analyze themselves

completely free, hand crafted



Project Vision

This project will allow Students to give mock tests/Exams and stimulate real Exam. Question banks can be uploaded into the Application by Teachers. Students can login/Register into the system and appear for exams. Whenever an exam starts, it cannot be paused/cancelled.

Students have to submit answers, and when the timer runs out, all the answers will be submitted. The System produces a very consised analysis report about the student's performance in the exam. Students can assess themselves by those reports and plan their strategy accordingly.

App features

 

Easy Exam Upload

The project allows very convenient exam uploads into the database. Not only the uploads are ACID, but also very user friendly.

 

Rich Exam Attemp

The user experience to attemp exam is very rich, not only it marks the questions on the go, it makes question list available at all times.

 

Analysis

Every user attemp is analysed in every possible way, and each user attemp helps to analysse other users in a more efficient way.
